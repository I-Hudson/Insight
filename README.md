# insight

Insight is an education project which I have been developing to advance my understanding of engine development.

## Features

Some of the features which are implemented are:
- Render graph – There is a render graph system in place in which each frame passes are added then reordered and finally executed. This system also manages its own resources.
- ECS (Entity Component System) – Use a system which allows component types to be updated in a cache friendly way.
- Reflection – There is reflection with using a custom tool which is based on the UE4 header tool.
